# 石田ぎがしー / Gigacee

## 👤自己紹介

- 企画、デザイン、プログラミングと、何でもこなすことのできるゲームクリエイターです。
- 2012 年に業界入りし、2015 年よりフリーランスとして活動しています。
- 少数精鋭でマルチスキルが求められるような開発現場が特に得意ですが、大規模チームでのリードやディレクションの経験もあります。
