# Gigacee

## 👤自己紹介

- 企画からデザイン、プログラミングまで、何でもこなすゲームクリエイターです。
- 2012 年に業界入りし、2015 年よりフリーランスとして活動しています。
- ゲーム開発に必要な工程は一通り経験してきていますので、1 人～小規模のチームでマルチスキルが求められるような開発現場が特に向いていると思います。
- 大規模開発現場のリード、ディレクションも経験あり。
